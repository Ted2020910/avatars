# Task 节点 — 明确任务与执行跟踪

Task 节点用于记录需要执行的具体任务。内容结构由任务本身决定，没有强制格式。

常见写法示例：

**任务描述**
用一两句话说清楚要做什么。

**完成标准**
什么情况下算完成？

**依赖 / 前置条件**
有哪些东西必须先准备好？

**执行记录**
过程中的关键决定或意外情况。

状态流转：pending → in_progress → completed
